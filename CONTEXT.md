# Project Context

## What it is
A single-file animated digital love letter micro-site (`index.html`). Designed to be opened from a text message link on a phone.

## Purpose
Replace a generic "I love you" text with something she actually feels. The entire experience is one emotional moment: a sealed envelope on screen, she taps it, it opens, a handwritten-style letter rises out.

## Goal
Make her heart melt the second she opens the link. Beauty and emotional warmth are the only success metrics. It should feel handcrafted, intimate, and unmistakably personal — not like a website, not like an app.

## Editing
The letter content lives in `index.html` between the two clearly marked `✦ EDIT YOUR LETTER CONTENT ✦` comment blocks. Change the salutation, paragraphs, and signature. Nothing else needs to touch.

## Stack
Plain HTML/CSS/JS. No dependencies. No build step. Works as a local file or hosted anywhere static (Netlify Drop, GitHub Pages, etc.).

## Design constraints
- Mobile-first, phone viewport only
- Palette: blush pink background, pink envelope flaps, rose wax seal, cream-white letter card
- No navbars, sections, footers, or any website structure
- WCAG AA contrast on all readable text
